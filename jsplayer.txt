■jsplayer 




■設置方法


[1] プログラムをダウンロードする
https://github.com/worldace/jsplayer/archive/master.zip


[2] jsplayer.js と proxy.php をアップする

※同じディレクトリにアップしてください


[3] jsplayer.js を読み込む。<head>内に記述してください

<script src="jsplayer.js"></script>


[4] <body> 内に次のようなscriptタグを記述して jsplayer() を実行する

<script id="player">
jsplayer({
    el: '#player',
    file: 'http://example.com/example.mp4',
});
</script>

この例だと、このscriptタグ自体がプレイヤに置換されます。



■起動オプション一覧

jsplayer({起動オプション});


---------------------------------------------------------------------------------------------------
el       : プレイヤを表示する場所をCSSセレクタかDOM要素で。このタグがプレイヤに置換されます
---------------------------------------------------------------------------------------------------
file     : 動画ファイルのURL
---------------------------------------------------------------------------------------------------
himado   : ひま動の動画ID。URLに含まれる文字です
---------------------------------------------------------------------------------------------------
key      : ひま動のkey。動画ページのソースを「name="key"」で検索して、そのvalueの値
         : これがないとコメント取得が1000件に制限されます
---------------------------------------------------------------------------------------------------
group_id : ひま動のgroup_id。 動画ページのソースを「name="group_id"」で検索して、そのvalueの値
         : これがないとコメント取得が1000件に制限される＋コメントが投稿できません
---------------------------------------------------------------------------------------------------
comment  : 表示したい独自コメントの一覧をJSON文字列で。文字列なので''で囲ってください
         : コメント一覧は [[本文,時間],[本文,時間],[本文,時間]] のような配列に入れJSONエンコードする
---------------------------------------------------------------------------------------------------
posturl  : 独自コメントの投稿先URL。POSTで送信されます
         : comment(本文)とvpos(時間)とfile(動画ファイルのURL)の3つの項目がある
         : 送信項目を追加したいなら、このURLに含めておこう
---------------------------------------------------------------------------------------------------
